# 📊 教师成绩分析工具

基于 **Streamlit** 构建的专业 Web 成绩分析平台，帮助教师快速完成考后数据分析。

---

## 🚀 快速启动

### 方式一：一键启动（推荐）
```bash
python run.py
```
脚本会自动安装依赖并在浏览器打开 `http://localhost:8501`。
### 方式二：手动启动
```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动服务
# 2.1 手动启动
    streamlit run main.py
# 2.2 脚本启动
    双击“执行.bat”
```

## 📁 项目结构

```
grade_analyzer/
├── main.py                   # Streamlit 主应用
├── run.py                    # 初始化脚本（自动安装依赖+打开浏览器）
├── 执行.bat                  # 一键启动脚本

└── assets/
│   ├── styles.css            # 自定义 CSS 样式
│   └── logo.png              # 学校 Logo（可替换）

```

## 📋 Excel 文件格式要求

| 列位置 | 内容     | 要求                     |
|--------|----------|--------------------------|
| 第 1 列 | 学号     | 非空字符串                |
| 第 2 列 | 姓名     | 非空字符串                |
| 第 3 列起 | 各科成绩 | 数值型，范围 0–100       |

> 文件大小 ≤ 10MB，支持 `.xlsx` 和 `.xls` 格式。

**示例格式：**
```
学号      姓名   语文  数学  英语  物理
2024001   张三    85    92    78    88
2024002   李四    72    65    90    71
```

## ✨ 功能特性

### 数据输入
- ✅ 支持 `.xlsx` / `.xls` 格式上传
- ✅ 10MB 文件大小限制
- ✅ 自动检测缺失值、非数字值、超范围分数
- ✅ 友好的错误与警告提示

### 数据分析
- 📊 **班级统计**：平均分、最高分、最低分、标准差、有效人数
- 📈 **分数段分布图**：六区间柱状图（含占比）
- 🕸️ **学生雷达图**：多科目均衡性可视化
- 🌡️ **成绩热力图**：一图览全班
- 📝 **成绩汇总表**：可搜索、按分数着色、支持 CSV 导出

### 报告生成
- 🖨️ 一键生成 PDF 格式成绩分析报告
- 支持自定义学校名称、班级、教师评价
- 支持上传学校 Logo 嵌入报告
- 报告包含封面、统计表、图表、学生表、教师评价

### 其他
- 💾 临时缓存持久化（时间戳 + 随机码命名）
- 🔄 会话内缓存避免重复计算
- 📋 完整操作日志（`app.log`）
